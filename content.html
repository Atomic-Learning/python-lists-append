The append method belongs to the list type in Python and is used to add a single item to the end of a list.

Basic append Usage

Create a list and add items to the end:

fruits = ["apple", "banana"] print(fruits) fruits.append("orange") print(fruits) fruits.append("grape") print(fruits)

append Modifies the Original List

Important: append doesn't return a new listâ€”it modifies the original list in place:

my_list = [1, 2, 3] result = my_list.append(4) print(my_list) print(result)

Return Value

append always returns None. As such, you do not normally need to capture its return value in a variable.
