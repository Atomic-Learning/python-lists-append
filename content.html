The append method belongs to the list type in Python and is used to add a single item to the end of a list. The original list is modified in place and no new list is created.

Basic Usage

When calling the append method on a list, a single argument is passed to it and that object is added to the end of the list:

fruits = ["apple", "banana"] print(fruits) fruits.append("orange") print(fruits) fruits.append("grape") print(fruits)

Return Value

append always returns None. As such, you do not normally need to capture its return value in a variable.
